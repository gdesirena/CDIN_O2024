Escalamiento¶

Qu es el escalamiento de los datos?: Es poner valores en el mismo rango o escala para que ninguna variable sea dominada por otra

Para qu nos sirve?: La mayora de las veces, los datos tienen variables que varan mucho en magnitudes, unidades y rango. Pero dado que la mayora de los algoritmos de aprendizaje automtico utilizan la distancia euclidiana entre dos puntos de datos en sus clculos, esto es un problema.

Las variables con magnitudes altas pesarn mucho ms en los clculos de distancia que las variables con magnitudes bajas. Para que esto no afecte tanto, debemos llevar todas las variables al mismo nivel de magnitudes. Esto se puede lograr escalando.

Algunos algoritmos de aprendizaje auntomtico donde el escalamiento importa son:

KNN: con la ditancia Euclidiana es sensible a magnitudes

PCA: este busca traer caractersticas con mxima varianza y la varianza es grande para variables con grandes magnitudes. Esto sesga el PCA a variables con grandes magnitudes

Gradiente Descendiente: se puede acelerar escalando. Ya que $\theta$ desciende ms rpido en rangos pequeos y eso hace que oscile menos ya que los brincos son ms parejos

Si estamos seguros que los datos son correctos, se puede realizar una transformacin para evitar la asimetra

Efectos de la asimetría¶

En datos asimtricos, la regin de la cola puede actuar como un valor atpico para el modelo estadstico y sabemos que los valores atpicos afectan negativamente el rendimiento del modelo, especialmente los modelos basados en regresin. Hay modelos estadsticos que son robustos a valores atpicos como los modelos basados en rboles, pero limitarn la posibilidad de probar otros modelos. Por lo tanto, es necesario transformar los datos asimtricos para que se acerquen lo suficiente a una distribucin gaussiana o distribucin normal. Esto nos permitir probar ms modelos estadsticos.

Frmula del ndice de asimetra esttica: $$skewness = \frac{\sum_{i=1}^{m}(x_{i}-\bar x_{i})^{3}}{(m-1)v^{3/2}}, v=\frac{\sum_{i=1}^{n}(x_{i}-\bar x_{i})^{2}}{m-1} $$

Podemos determinar el nivel de asimetra de la forma:

  • Distribucin altamente sesgada: si el valor del sesgo es menor a -1 o mayor a +1.
  • Distrubicin moderadamente sesgada: si el valor del sesgo es entre -1 y -0.5 o entre +0.5 y +1
  • Distribucin aproximadamente simtrica: si el valor del sesgo est entre -0.5 y +0.5

def dqr(df):
    #%% Lista de variables de la base de datos
    columns = pd.DataFrame(list(data.columns.values), columns=['Nombre columnas'], index=list(data.columns.values))
    
    #%% Calculo de la asimetra
    skewness = round(df.skew(),2)
    skewness = skewness.to_frame()
    skewness = skewness.rename(columns={0: "skewness value"}) 

    def f(x):
        if x['skewness value'] < -1 or x['skewness value'] > 1: return 'Highly Skewed'
        elif (x['skewness value']<=0 and x['skewness value']>=-0.5) or (x['skewness value'] >=0 and x['skewness value']<=0.5):
            return 'Symmetric distribution'
        else: return 'Moderately skewed'

    skewness['skewness'] = skewness.apply(f, axis=1)
    return columns.join(skewness)

¿Cómo mitigar la asimetría?¶

Box Cox: Una transformacin de Box Cox es una transformacin de variables no normales en una forma normal. (Lambda entre -5 y 5). Aplicada a valores positivos.

$$y(\lambda)= \begin{cases} \frac{(y)^{\lambda}-1}{\lambda} & \text{if $\lambda\neq0$}\\ \log(y) & \text{if $\lambda=0$ }\\ \end{cases}$$

Datos con sesgo positivo¶

Transformacin de raz cbica: Es una transformacin bastante fuerte con un efecto sustancial en la forma de la distribucin, pero es ms dbil que el logaritmo. Tambin se puede aplicar a valores negativos y cero. $$x_{i}^{*}=(x_{i})^{1/3}$$

Transformacin de raz cuadrada: Aplicado solo a valores positivos. $$x_{i}^{*}=\sqrt(x_{i})$$

Transformacin logartmica: Es una transformacin fuerte que puede ser usada para reducir el sesgo positivo. Aplicado solo a valores positivos $$x_{i}^{*}=\log(x_{i})$$

Datos con sesgo negativo¶

Las transformaciones comunes incluyen elevar al cuadrado, raz cbica y logartmica.

Transformacin al cuadrado: Tiene un efecto moderado en la forma de la distribucin y podra usarse para reducir el sesgo hacia la izquierda. $$x_{i}^{*}=(x_{i})^{2}$$

Otro mtodo para manejar la asimetra es encontrar valores atpicos y posiblemente eliminarlos.

Valores Atípicos¶

Un dato se considera atpico si aparece por fuera de un rango que se considera vlido o aceptable para el fenmeno que se analiza.

Para determinar si los datos tienen valores atpicos es posible usar grficas de bigotes en los datos.

Para cuantificar el rango aceptable o permitido para encontrar datos atpicos se utiliza el parmetro IQR (inter quantilerange).

$$IQR=75^{th}quantile - 25^{th}quantile$$$$upper_{boundary}=75^{th}quantile+(\alpha*IQR)$$$$lower_{boundary}=25^{th}quantile-(\alpha*IQR)$$

donde $\alpha$=1.5, y est relacionado a la probabilidad de ocurrencia si los datos tuvieran una distribucin normal.

#Funcion para determinar ms especficamente los outliers a travs del rango IQR

def find_boundaries(df_var,distance=1.5):
    IQR = df_var.quantile(0.75)-df_var.quantile(0.25)
    lower = df_var.quantile(0.25)-IQR*distance
    upper = df_var.quantile(0.75)+IQR*distance
    return lower,upper

    outliers_df = data.loc[outliers, col]

Tranformación de datos atípicos¶

Si los datos atpicos son importantes y no es posible descartarlos, entonces es posible hacer una transformacin de los datos para que se encuentren en un espacio con escalas similares.

Una transformacin que se puede hacer es utilizar una funcin que mitigue la asimetra.

Datos Nulos¶

Un dato nulo se refiere a la ausencia de un valor o valores en al menos una de las variables. El origen de estos valores pueden ser distintos (errores de captura, censura, funcionamiento del fenmeno, etc.), pero hay que tratar de identificarlas para poder realizar el tratamiento correcto.

Si los datos no pueden ser eliminados y es necesario sustituirlos para poder procesarlos con algoritmos sensibles a la falta de datos. Dentro de los criterios comnmente usados para sustituir los valores nulos se encuentran.
